# Ray Tracing-Assisted Spotlight SAR for UAVs in Reflective Environments
Master's Thesis on Ray Tracing-Assisted Spotlight SAR for UAVs in Reflective Environments - Assessing the Influence of Pose Stability on Image Formation

## About The Project

This is a Mechatronics master's thesis study, done in collaboration with KTH Royal Institute of Technology and AFRY AB in Stockholm, Sweden. The study is an experimental, simulation-based investigation of how altitude and range disturbances affect image formation in low-altitude UAV-based SAR image formation. The SAR signal simulation is integrated with a Ray Tracing-based propagation model using SBR. The simulation takes a .osm map to create a point cloud of the target scene, generates an LFM pulse and uses the propagation model to track the radio waves and calculate the received signal characteristics. The image formation method used is RMA.
